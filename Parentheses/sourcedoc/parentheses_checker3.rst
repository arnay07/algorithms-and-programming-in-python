Ce module comprend la fonction qui, si le fichier est mal parenthésé,
renseigne l'endroit où se trouve le mauvais parenthésage; cette fonction prend en compte
le fait que le mauvais parenthésage se trouve dans les commentaires (dans ce cas elle ne fait rien).


Questions
---------
Le fichier bad_stack3.py est syntaxiquement correct en python

Le mauvais parenthésage se trouve dans la docstring

Si le mauvais parenthésage se trouve dans la docstring ou dans un commentaire l'interpréteur python 
pourra exécuter le programme sans problème
