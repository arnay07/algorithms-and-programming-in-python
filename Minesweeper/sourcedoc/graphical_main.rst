Le jeu
======

Pour jouer de manière graphique, il faut taper dans le terminal, une fois dans le dossier src:

.. note:: python3 graphical_main.py [width] [height] [nbombs]

Les paramètres sont optionnels, si on ne les  met pas, le jeu s'initialise avec une largeur de 8, une hauteur de 8 et 5 bombes
