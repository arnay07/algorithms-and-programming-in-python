    Cards are defined by a value and a color.
    Possible values and colors are listed in ``Card.VALUES`` and ``Card.COLORS``.

    >>> c1 = Card("Ace", "heart")
    >>> c1.get_color()
    'heart'
    >>> c1.get_value()
    'Ace'
    >>> c1
    Card("Ace", "heart")
    >>> c2 = Card.random()
    >>> c2.get_value() in Card.VALUES
    True
    >>> c2.get_color() in Card.COLORS
    True
    >>> c1 == c1
    True
    >>> c1 != c1
    False
    >>> c1 < c1
    False
    >>> c1 <= c1
    True
    """
